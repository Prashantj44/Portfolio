Welcome to the source code of my personal portfolio website! 

This is a premium, high-performance static website designed to showcase my projects, skills, and experience as an **AI & Machine Learning Engineer**. It features a modern dark-mode aesthetic, dynamic grid layouts, and smooth micro-animations.

## 🛠️ Tech Stack
- **HTML5**: Semantic and accessible structure.
- **Vanilla CSS3**: Custom CSS variables, CSS Grid, Flexbox, and premium typography (Inter font).
- **Vanilla JavaScript**: Lightweight interactions, mouse-tracking radial glow effects, and staggered animations.

## ✨ Features
- **Dark-Mode UI**: High contrast, elegant color palette `#030303` background with soft `#ededed` text.
- **Dynamic Mouse Tracking**: A subtle background glow that follows the user's cursor for a premium interactive feel.
- **Staggered Animations**: Smooth load-in fade animations for project cards and sidebar sections.
- **Responsive Layout**: Adapts flawlessly across desktop, tablet, and mobile viewing.

## 📥 How to Run Locally
Since this is a purely static website, no complex build tools are required!
1. Clone the repository: `git clone https://github.com/Prashantj44/portfolio.git`
2. Navigate into the folder: `cd portfolio`
3. Simply open `index.html` in your favorite web browser!
